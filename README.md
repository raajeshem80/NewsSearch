# NewsSearch
## NewsSearch API with Spring Boot and Angular JS.

### Technology stack:

* Maven;
* Tomcat embedded;
* Spring Boot;
* JUnit;
* Lombock;
## Build

Build the application:

```bash
mvn clean package
```

## Run

Run the included docker-compose file with:

```bash
docker-compose up
```

## Swagger

### Swagger UI end point

http://localhost:8080/swagger-ui.html


## To Stop

Run the included docker-compose file with:

```bash
docker-compose down
```
